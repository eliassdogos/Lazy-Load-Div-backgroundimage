Small Snippet for loading images into Div background after page load

1. Set classname for div class . E.g lazybg in your file html or php. (view.php)
2. Include divload.js file 
3. In view.php set the final image to be visible after page load in data-pic="background-image: url(finalimage.png);" 
4. Include loading.gif image to view before page loading

#jQuery v1.12.4 required
